# League signup how-to
The VESA league uses a Discord bot for signups. This helps us expedite our sorting process by eliminating Discord username misspellings and other discrepancies while organizing, and reduces the time we spend assigning roles during placements.

Please provide accurate information on your skill level, comp experience, and VESA experience so we can sort and prioritize you correctly for placement matches. Not only does it hurt the whole lobby (57 other real human beings!) when inaccurate information is provided, it also hurts your ability to place into an appropriate division.

**Please read the following how-to so you don't spend unnecessary time making mistakes during your signup!**

If you're here because you didn't read and you're having issues, read the steps below and then check the "Something went wrong" section to help you get signed up.

## How to use the /league-signup command

* In the #league-signup channel type "/league-signup"
  * While you type you will see a drop down of options of commands to run, select the newt-bot league signup command
  * ![Selecting user](/usage/assets/league/selecting-user.png)
* Fill out all the required fields
  * team name
    * Type your team name
  * player 1 (captain)
    * Start typing the name of your captain and select the correct user from the dropdown. The captain **MUST** be on the team
    * ![Selecting player](/usage/assets/league/selecting-user.png)
    * ![Successfully selected player](/usage/assets/league/selected-user.png)
  * player 2
    * Start typing the name of player 2 and select the correct user from the dropdown
  * player 3
    * Start typing the name of player 3 and select the correct user from the dropdown
  * comp experience
    * Type your scrim frequency/experience & main servers for your team. Ex: "~2 days per wk, for 2 yrs, EEC"
  * player1-rank
    * Select from the dropdown the ranked tier this player achieved in the latest ranked split or most recent season with a decent amount of games played
    * ![selecting rank from dropdown](/usage/assets/league/selecting-rank.png)
    * ![successfully selected rank from dropdown](/usage/assets/league/selected-rank.png)
  * player1-s12-vesa-division
    * Select from the dropdown the previous VESA division this player played in, select "None" if you did not play or all you did was sub and you were not on a roster
  * player1-overstat-link
    * Paste the overstat url of this player or write "None". If this player has already used the /link-overstat command in the scrim server you can also write "None" and it should autofill
  * player1-platform
    * Select from the dropdown the platform this player plays on
  * player2-rank
    * Select from the dropdown the ranked tier this player achieved in the latest ranked split or most recent season with a decent amount of games played
  * player2-s12-vesa-division
    * Select from the dropdown the previous VESA division this player played in, select "None" if you did not play or all you did was sub and you were not on a roster
  * player2-overstat-link
    * Paste the overstat url of this player or write "None". If this player has already used the /link-overstat command in the scrim server you can also write "None" and it should autofill
  * player2-platform
    * Select from the dropdown the platform this player plays on
  * player3-rank
    * Select from the dropdown the ranked tier this player achieved in the latest ranked split or most recent season with a decent amount of games played
  * player3-s12-vesa-division
    * Select from the dropdown the previous VESA division this player played in, select "None" if you did not play or all you did was sub and you were not on a roster
  * player3-overstat-link
    * Paste the overstat url of this player or write "None". If this player has already used the /link-overstat command in the scrim server you can also write "None" and it should autofill
  * player3-platform
    * Select from the dropdown the platform this player plays on

If everything went right the response should look like this:
![success image](/usage/assets/league/signup-success.png)


## Something went wrong?

Don't worry — you don't have to retype everything. Simply click on the little blue league-signup button on the line that reads "[your username] used league-signup". On mobile that will open a little popup where you can choose to open the command and then fix any of the things that you entered incorrectly, on desktop it'll open a little popup where you can copy paste and fix them that way.


Error:
![Command error](/usage/assets/league/overstat-error.png)

After clicking the blue button:
![Command error popup](/usage/assets/league/error-popup.png)

After clicking open command:
![Redo command modal](/usage/assets/league/signup-modal.png)

Fix your errors and click send.

Still can't sort it out? Open a ticket and let the admins know what's going wrong — screenshots of the errors will greatly help.
